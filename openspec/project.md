# Project Context

## Purpose
The "Jakarta Dental Exhibitions 2026" (JADE 2026) website serves as the official digital platform for the event organized by World Kpop Center Indonesia (WKCI). It provides event information, attendee registration, and exhibitor details.

## Tech Stack
- Laravel 12
- Livewire 4
- Tailwind CSS
- daisyUI
- MySQL/PostgreSQL

## Project Conventions

### Code Style
- PSR-12 for PHP.
- Tailwind CSS utility-first classes for styling.
- Livewire components for interactive UI elements.

### Architecture Patterns
- Standard Laravel MVC architecture.
- Livewire components for dynamic frontend interactions.

### Testing Strategy
- PHPUnit/Pest for unit and feature tests.

### Git Workflow
- Feature branching strategy.

## Domain Context
JADE 2026 is a major dental industry event in Indonesia, bridging professional dental exhibitions with the event management expertise of World Kpop Center Indonesia.

## Important Constraints
- Performance: Fast loading times for the homepage.
- Design: Modern, professional, and visually appealing using daisyUI.

## External Dependencies
- None (Local assets from `assets/` folder will be used).
