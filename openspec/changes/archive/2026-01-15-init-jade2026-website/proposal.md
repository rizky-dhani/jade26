# Proposal: Initialize JADE 2026 Website

## Why
To provide a professional and interactive web presence for JADE 2026, enabling information dissemination and future attendee engagement.

## What Changes
- Establish the foundational Laravel + Livewire project structure for the Jakarta Dental Exhibitions 2026 (JADE 2026) website.
- Initialize Laravel 12 project with Livewire 4.
- Configure Tailwind CSS and daisyUI.
- Create a responsive homepage with Hero, About, Organizer, and Contact sections using local assets from the `assets` folder.

## Impact
- Affected specs: `core` (new capability).
- Affected code: Root directory structure, `resources/views/livewire/home.blade.php`.
